== Description ==
Sync stock between WooCommerce and dc-garment.com/staff/.  
- Updates WooCommerce stock via REST API.  
- Reduces stock in external inventory when an order is placed.  
- Configurable API URL in the WordPress settings.

== Installation ==
1. Upload the plugin folder to `wp-content/plugins/`
2. Activate the plugin in **WordPress Admin → Plugins**
3. Configure the API URL in **Settings → Stock Sync**
